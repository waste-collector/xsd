# xsd

The schema includes the following main types:

- **User** (abstract)
- **Citizen**
- **Employe** (abstract)
- **CollectAgent**
- **Administrator**
- **Vehicule**
- **CollectTour**
- **IncidentReport**
- **TourReport**
- **Zone**
- **City**
- **AnnuelReport**
- **AnnuelObjectif**

Each XSD type corresponds to a class in the UML model.
